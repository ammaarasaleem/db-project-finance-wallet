# Frontend (Vite + React + TypeScript)

## Run locally

1. Install dependencies:

```bash
npm install --legacy-peer-deps
```

2. Start dev server:

```bash
npm run dev
```

Default URL:

- http://127.0.0.1:8080

If localhost does not open reliably in Chrome on macOS, use:

- http://127.0.0.1:8080

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Tests

```bash
npm run test
```
